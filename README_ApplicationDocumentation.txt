This shiny application allows the user to predict the car mileage given the 
inputs.

The user is allowed to specify the car's
1. Number of cylinders
2. Displacement
3. Horsepower
4. Weight

The shiny server uses R Studio's mtcars dataset to model the mileage using 
random forest algorithm.

The predicted mileage is displayed in RED on the web page. The input values used 
for the prediction are also displayed.
